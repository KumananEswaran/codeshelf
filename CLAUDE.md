## Commands

- **Dev server**: `npm run dev` (runs on http://localhost:3000)
- **Build**: `npm run build`
- **Production server**: `npm run start`
- **Lint**: `npm run lint`
- **Test**: `npm test` (single run), `npm run test:watch` (watch mode)
- **Test coverage**: `npm run test:coverage`
